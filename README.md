##Run a Simple Web Server --> [Server Basics](http://www.2ality.com/2014/06/simple-http-server.html)
You can run a Simple Web server in your Terminal on a Mac by:(**Python Might Be Required**)
```
cd Desktop
cd NameofDirectory
```
##Python Example:
```
Python -m SimpleHTTPServer 
```
